# Mobile Data Analysis Dashboard (Power BI)
This project is an interactive Power BI dashboard designed to analyze mobile sales data and provide meaningful business insights.

# 🎯 Objectives
- Analyze overall sales performance
- Track monthly sales trends
- Understand customer behavior and ratings
- Identify top-performing brands and mobile models
- Compare different payment methods

# 📈 Key Metrics
- Total Sales: 769M
- Total Quantity Sold: 19K
- Total Transactions: 4K
- Average Sales Value: 40K

# 📊 Dashboard Features
- 🌍 Sales by City (Map visualization)
- 📅 Monthly Trends (Line chart)
- ⭐ Customer Ratings Analysis
- 💳 Payment Method Distribution
- 📱 Top 3 Mobile Models
- 📊 Sales by Day Name
- 🏷 Brand-wise Performance Table
- 🎛 Interactive slicers for filtering data

# 🛠 Tools & Technologies
- Power BI
- DAX
- Excel (Data Source)
- Data Modeling
